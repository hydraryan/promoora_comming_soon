# Promoora Coming Soon Page

This is a simple "Coming Soon" landing page for Promoora, scheduled to launch on March 5th, 2026.

## Project Structure

- `index.html`: The main HTML structure.
- `style.css`: Styles for the dark theme, typography, and layout.
- `script.js`: Logic for the countdown timer.

## How to Run

Simply open `index.html` in your web browser.

## Customization

- **Date:** To change the launch date, edit the `launchDate` variable in `script.js`.
- **Text:** Edit the headline and subheadline in `index.html`.
